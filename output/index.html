VirtualBox in arch

二 30 七月 2013

tags: archvirtual boxvagrant

arch跑vbox时显示： 

VirtualBox is complaining that the kernel module is not loaded. Please
run `VBoxManage --version` or open the VirtualBox GUI to see the error
message which should contain instructions on how to fix this error.

按提示输入VBoxManage --version: 

WARNING: The vboxdrv kernel module is not loaded. Either there is no module
    available for the current kernel (3.10.3-1-ARCH) or it failed to
    load. Please reinstall the kernel module virtualbox-host-modules or
    if you don't use our stock kernel compile the modules with
    sudo dkms autoinstall
    You will not be able to start VMs until this problem is fixed.
4.2.16_OSEr86992

尝试升级内核： 

sudo pacman -Ss linux-headers  
sudo pacman -Ss virtualbox-host-modules

还是英文的arch wiki给力，发现是内核还要手动load： 

sudo modprobe vboxdrv
sudo systemctl enable dkms.service

然后就OK了。

    一直很疑惑，为什么这个板块要叫做“日记”呢？
    日记不是给自己看的吗，日记不是留下回忆的吗，日记不是私藏秘密的吗？
    那为什么要公开呢，为什么要评论呢，为什么要回应呢？

    或许是人太寂寞了吧，他想找个角落吐露自己的心声，很小心，但有想有倾听者，很惆怅。如娇羞的小鹿，离你远远的，但有在那边注视者你，享受着你的关心。
    又或者人是自恋的动物，他想被人知道，他想被人关心，他想被人抚摸。于是，他像个孩子般，蹑手蹑脚的把拙作藏在身后，当着大人的面。他知道这会吸引他们的注意力，又有借口掩饰自己的小心思。
    又或者是自怜，又或者是宣泄，又或者是无心……

    犹记那年年少轻狂，恨不得全世界都认识自己，那时日记是记在blog上，那时蠢得可笑。

    再后来，把他封印在心里，因为学会了孤独，学会了坚强。
