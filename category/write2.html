选择 一

最近在看《影响力》，书中提到了决策问题，这一直是我头疼的问题。
对于完美主义拖延症晚期患者来说，特别是二选一最为纠结的。当掺杂太多因素时，是“高瞻远瞩”还是“活在当下”，将不再由理性来判断。 为了生成一个“主见”，我决定给自己找一套“抉择”的标准，以免日后犹豫。

昨天看了一个帖子，列举程序猿们的开发工 具，想想自己这一年内为开发工具也做了 很多抉择，就列举出来作为判断依据吧：

开发工具

  • OS： Windows -> Ubuntu
  • Editor: Vim -> Sublime Text -> Emacs
  • Version: SVN -> Git
  • Terminal: Gnome Terminal -> Guake
  • Shell： bash -> zsh
  • Broswer: Firefox -> Chrome

Other

  • Blog: Wordpress -> Hexo
  • IT Skill: Android -> Web

    对工具的选择到没太多的思索，基本是随着技术的提升学会了更好的东西吧，就像SVN到Git， Bash到Zsh，都只是掌握后有了更高的需求而作出改变。 但一些其他的东西，却也有不同判断依据：

活在当下

虽然“活在当下”近年来被很大一部分人接受，但我还是没法认同它，可能我属于激进派吧， 还不想这么快就放弃攀登。而这反而和”和谐“有异曲同工之妙，更像执政党的维稳手段。

但每次做选择时，我都不得不考虑它。就像操作系统，以前我一直用Windows，因为我主要 做Android开发，JAVA和Eclipse有很好的跨平台性，我完全用不到Linux的“优点“，而 Windows又能给我提供更好的桌面体验，除了装B，我还有什么理由选择其他操作系统呢？但 后来Android无法满足我了，我开始学习Web，这时蹩脚的Windows完全无法满足的我的需求， 于是现在，我变得无法离开Linux了。

“任何真理都是合理的偏见”，在不同的境况下，为何不作出不同的选择。团队有个女生说 我太善变了，因为在给她推荐手机时第一天说“性价比高的东西，不值得买”，第二天我就 推荐她买小米了,我很无辜。当预算在3000——4000时，品质当然是第一选择（特别是在你摔 坏了N部手机后），当预算在2000左右时，我想小米是无二的选择了。

中国人讲究天时地利人和，大概就是如此吧。

高瞻远瞩

“高瞻远瞩”算是小时候老师提得最多了的吧，但有时候却会陷入“好高鹜远”的漩涡。

以前在code之前，一定会先想用什么技术、架构，总考虑后期数据量大了、或者需求改了的 时候怎么办，一定要采用最好的技术，既是是一个小项目。

记得刚学web时还特别反感动态语言，认为其性能不如静态语言，理由很简单，Java处理数 据库的速度比PHP高一个数量级，安全性也更好，君不见Facebook前期偷懒，后期不得不把 PHP完全Hack了才能满足高并发。Twitter开始用Ruby很爽吧，现在后悔了吧。

为了方便，在选 动态语言时也考虑了很久，Pyhon还是Ruby，最后还是选了性能更好的 Python。Ruby虽然在特性上更符合我的性格，但ROR确实没在性能上很乏力。

作出了这样的选择后，可想而知我当年是有多痛苦，别人用php一下午搞定的课设，我用 J2EE这反人类的玩意写了一个星期，捉急！！！

后来看了一篇某高人的文章，说我这种做法是“自寻死路”。一个菜鸟的项目，用户量能有 多大，并发能有多高？盲目的最求技术却输在了时间上，这是互联网世界中不能忍的。现在 的硬件完全能够弥补动态语言带来的性能缺失，与其说是Google的V8拯救了javascript，倒 不如说是现在的机器跑得动Chrome了。（建议和我一样对动态语言耿耿于怀的朋友去看下 《黑客与画家》）

虽然容易好高鹜远，但远见还是必须的。
就像Blog，去年Octopress特别火时我便对它没什么好感，就一个原因——Ruby，Ruby的执行 效率一只是个问题，与其用Ruby来做静态生成器，我宁愿拿等待的时间来用C写一个。
果不其然，当年换Octopress的人也开始纷纷转型了，如果你有100篇博文，每次生成你都要 等几分钟，然后还要花更多的时间去把它pull到github上，一个字，“累”。
忍受了半年Wordpress，看到Lucifr也换成了Hexo，一个Nodejs的生成器，在他出到没什么 bug的版本后，我也迁移了。Nodejs的速度果然不是盖的。

个人认为Octopress和Ruby都属于无法经受时间考验的东西，初期用着很爽，后期就要为它 付出代价了（怎么感觉和.Net一样。。。）。

对于从Android转型Web我也抱有相同的态度，Android就像一个大框架，限制太多了，而就 其前景来说，诺基亚的倒塌已经证明了Android也不会走得太远。基于Html5的Firefox OS和 更为底层的Ubuntu Phone在初期可能无法撼动Android，但他们会走得更远是可以预料的。

而对于Web来说，刚接触我就被它的自由吸引了，更多的语言，更多的技术，更多的框架， 如果说Android是刚开坑出来的金矿，那Web就是无尽的海洋，孕育了一切的奇迹。

————节外生枝了好多————

技术上的抉择只是这一年来的一小部分，但把他写出来，确实也让重新审视了不少东西。每 次写随笔总是想到什么写什么，写到最后，主旨早就变了。

我想对于”抉择“，我会所有理由一个权值，然后再来排序，选出最后的结果。
现在看来，”活在当下“应该分配更多的点数了。

这个世界总是充满了惊喜，太早的为未来做打算，可能不是一个好的选择，还是先找到自己 的位置比较重要。

写在第一次规划”抉择“。

起源

引子——

 

你是谁？

你从何而来？

你要到哪去？

 

这不是哲学家的问题，这是我们每个人的问题。

当你思考自己的起源时，都会陷入无限的循环与苦楚之中。

 

我来自哪里，我为何而来，谁创造了我？

如果是上帝，那谁又是上帝，上帝又从何而来？

如果是宇宙大爆炸的尘埃，那宇宙又从何而来，宇宙外面是什么，外面的外面又是什么？

 

或者说上帝是永恒的，宇宙是无限的，那永恒与无限又是什么，这不是一个合理的解释

至少我无法接受。

 

这个问题让我那难以入眠，每当想起，都焦躁不安

世界就像魔术师手里的兔子，诞生于诡异的帽子魔法

我们就诞生于兔子的毛发上，想努力攀爬到顶端去看看魔法师的样子，想看清这不解的魔法。

 

但我们失败了，年长时，我们接受的“现实”，接受了“真理”

于是世界变成了：本来就是这样

我们遗弃了本源，在自己的秩序里生存，在终焉里安眠。

 

但还有一群人不接受这样的现实，他们保留了那原始的疑惑

他们在探寻着自己的起源

他们在宗教、科学、魔法、哲学中挣扎

我称他们为开辟的使者

 

而我，则默默的记录着这一切。

 

  疑惑——

 

起源的问题，已经在人类浩瀚的历史里被探讨了多次，但没有人得出了结果，没有人能够用双眼去见证它，他的一切都被“混乱”所掩盖。

试着跟着我做，闭上眼，在大脑中构建一个宇宙，我们的星云置身其中，我们的星球在银河系里，而我们也只是星球上不起眼的一点，然后将视野放大，放到宇宙的边缘，然后是宇宙之外。

如果宇宙是有空间大小的，那他的外面一定还有东西，是另一个宇宙，父级宇宙的外面呢，如果他也有大小，那外面一定还有。继续放到，将思维无限放大，去试着包容这一切，让大脑去解析所有的宇宙。

于是混乱开始了，这没有终点，当你记录了一定数量的宇宙后，你开始焦躁，开始迷惑，开始无法冷静，手心出汗，双腿颤抖，你的内存已经泄漏，栈溢出是必然的结果。

 

好， 我们换个假设，如果宇宙之外不是无限的，我们定义我们能理解的宇宙为“有”，那么我们不能理解的必定为“无”了。我们生活在“有”里，有的外面是“无”。 好的，悖论出来了，“无”在“有”之外，即“有”之外还有东西，因为“有”的定义是有范围的，而范围又是需要名为“无”的“有”来确立。我们无法定义 “无”。

 

那么世界是起源于“无”吗？

无数的物质世界在大爆炸里从“无”诞生。

 

科学家为了定义这个世界曾经作出如下假设：

能量 == 物质 + 反物质

 

认为宇宙中还有一些称为“反物质”的粒子，当他们与物质想碰时，便会与之一同湮灭，产生能量。至今反质子、反中子还没被找到。

如果假设成立，那么我们便需否决“无”，或者“无”就是能量。

 

====

有限或无限，无或者有都无法让我解释这个世界，以及这个世界以外的世界。

当我去竭力思考时便会陷入一种“混乱”，让我无法继续，如果继续下去，我想我的大脑会因无法接受这海量的数据而变得疯狂。

 

于是我开始从已有的“猜想”中开始寻求解脱。

 

 

体系神话——

 

当人类无法理解某些因果时，就会用一些荒唐的故事来神化它们

比如闪电，天空最强大的自然现象，每一次劈开天空，都震撼人心

于是他被赋予给最强大的幻想，宙斯、佐尔，这些最强大的神明才有驾驭它的能力。

 

现代，也有无数学者在遇到无法解决的问题时开始了幻想之路，比如克苏恩神话。他给我带来了很多不错的思路，结合人类历史上的种种创奇，我总结了自己的神话体系。

 

混沌的开始——

 

世界依旧起源于无，一片空虚，一片寂静

 

但这是矛盾的，这矛盾在世界里膨胀，不知从何时起，不知在哪个维度出现

只知道那是自世界诞生，或者说世界有了该存在的意识起

在每一个角落，或者就是整个空间里

都开始争吵

 

矛盾越来越来

越来越大

 

“轰”

寂灭中出现一声翻天覆地的“巨响”

 

宇宙产生了。

 

宇宙产生自这无法解决的矛盾

因为这矛盾宇宙产生

 

宇宙就是悖论 宇宙源于悖论 宇宙带来悖论

混乱与秩序——

 

想想我们自己，我们是什么

是一堆有机物的组合，是肉体与灵魂的交杂，是物质世界的随机因果？

 

不，我们是矛盾，是混乱与秩序的矛盾。

看看我们自己，拥有理性，却也具有感性

我们可以像机器一样生存，遵循没一个因果，在没一个公式下的推导下按照理性的逻辑进行“必然”的下一步;

我们也可以像梦境一样或在幻想里，把激素与神经的共同作用称为爱情，把复杂的反射机制归为愤怒、悲伤和高兴，在感性里迎接未知的未来。

我们把理性称之为“秩序”，把感性称之为“混乱”

而我们无数次亲身体会到了两者的碰撞，无数次艰难的在两者间选择

谁是谁非，总是来自于“真理”——那合理的偏见

 

微小的我们是矛盾体，世界也是矛盾体。

 

神——

 

世界爆炸了，诞生了宇宙，也诞生了两个认知意外的“物质”

秩序与混沌

 

他们分属矛盾的两边，是世界无法承受他们时通过大爆炸释放的。

大爆炸后他们交错在一起，却互相排斥，挤压

于是宇宙越来越大，温度越来越高

 

高温让他们的结构变得不稳定，他们开始分裂，变成无数粒子，散落在宇宙里

慢慢的，他们的意识开始消亡，宇宙开始冷却

 

冷却中无数秩序粒子组合在一起，形成了宇宙里的尘埃

混乱也开始汇聚，但混乱的特性让他们无形

 

秩序更为稳定，他们在无数随机或者说“必然”下，重新获得了意识

成为无机的神——泰坦

无情、强大、神秘、中里的东西

 

他们诞生目的只有一个，修复宇宙，让世界重新获得秩序，归于无。

 

混乱也开始冷却，附着于没有意识的尘埃上，开始展现自己的生命意识。

他们被称之为“古神”，混乱的结合体

他们没有形体，他们唯一的意识就是吞噬

就像癌细胞，他们在宇宙里肆虐，吞噬遇到的一切，包括泰坦。

 

显然泰坦们力量更为强大，遵循秩序让他们更适合在物质世界生存。

他们组成了议会，他们开始合作，开始操纵尘埃，将他们做成武器，开始毁灭古神。

 

但是和他们一样，古神起源于矛盾，“混乱”是无法消除的“存在”

所以泰坦只好将古神封印，用更多的物质建立监牢，让古神沉睡于中。

 

而这监牢，就如同我们可以理解的尘埃汇集物——星球。

 

苏醒——

 

古神们陷入了深眠，但也有例外。

在一颗星球上，一个古神开始苏醒，他发现监牢在周边星球的引力下出现了裂口。

他开始伸展自己的思维，用混乱去影响周遭。

特别是泰坦留下的无机守卫。

 

古神的神格就像海洋包围者这些由颗粒构成的守卫，就像在电场下光速震颤的电子，开始变得不稳定。

守卫的结构被打乱，粒子们不断汇聚与散开，不断重组又不断被撕裂。

 

最后，守卫们被打败了，变成了古神的奴仆——“上古之神”

在这颗星球上一共诞生了四位上古之神

上古之神由于源自被吞噬的守卫，他们在物质世界有了形体。

上古之神不断的吞噬物质并且开始疯狂的生长，但缺失秩序的他们只能像肉团一样，不断的生出触须，粘液

 

战乱——

 

泰坦发现了古神的异动，降临到这可星球，开始整顿并毁灭这的一切。

他们在地心遏制着古神，并放出更多的守卫去消灭地表的上古之神。

 

显然泰坦低估了上古之神，守卫的力量不足以形成压倒性的优势。

吞噬了守卫的上古之神也开始有了意识，秩序影响了他们的本源

他们的生长改变了星球的环境，甚至使这颗星球出现了大气。

被守卫斩断的肢体在分解后形成了新了生物——有机体。

那些有机体脱离的古神的控制，又结合了守卫的秩序，在两种本源下开始重新组合。

 

最终，古神再次被打败了，又一次陷入的长眠。

当泰坦们回到地表时，他们惊愕了

持续几个世纪的战斗后，这颗丑陋贫瘠的星球发生的翻天覆地的变化。

 

上古之神被守卫封印到了地底

而地表却出现了不属于两者的生物

他们拥有混乱一样生命，却有秩序一般的外表

 

他们的生长遵循这秩序，行为却无法被理解。

泰坦们无法定义他们，不知道是否该给予毁灭。

如果是守卫，他们拥有自己的程式，泰坦可以轻易推导出其行为方式

如果是上古之神，泰坦们无法检测到秩序时只需要给予毁灭

而现在，他们不知所措

 

议会最终决定，降临到生物头上的裁决是“观察”

他们留下更佳强大的守卫监视着这些生物

如果其进化为混乱，则立即将其毁灭

 

然后，泰坦们离开了。

我的2011

恍然中，就这样一个人走到了世界末日，回忆过去，点点滴滴，确如云烟，淡然彷徨。

这个时候回忆2011确实有点晚了，没办法，我就是这样一个懒散的人，有兴趣在脑海中描绘，却无心思在现实中构建。或许感动只属于自己，但也容易忘记，于是就在末日的前夕，做一份备份吧。

2011我是如何走过的呢？

呃啊— 已经记不清了，或者说真的不想去回忆，90后都拥有逃避的属性。

大一下到大二上是人成熟最快的时候吧，也是从中二到大叔进化的跳板，梦想变得模糊，高贵的刺也被扒光。我也没能逃脱命运的安排，经历了很多痛，然后消沉，逃避，堕落——

一

应该从退Dian团队谈起吧，那段日子真的很难过。刚开学时，奶奶就走了，第一次尝到了失去亲人的痛，在悲伤中人也变得浑浑噩噩。迷迷糊糊中，作为新人加入了点团队的华三组，也毫无斗志的开始了学习。或许我属于疯子吧，更喜欢挑战和激情。简单无谓的软件测试让我毫无动力，刘老师井然有序的管理让我想起了初中最痛恨的教导主任（不是我吐槽 她们俩长得真像），于是每天就麻木的背着电脑去看RFC，写点东西，然后有索然无味的回来。一切就是这般过去了。

这个时候我又想学习下了，大一加权没好好搞，勉强80。咱也不是智商低，大学这点东西想好好弄还是有信心的，奈何这种快速学习，并且连续性不强和课程实在让人提不起劲，于是旷课，随懒觉，抄作业一并归来。不过有拿的出手的东西，自认为技术还行，软件硬件都摸得到边，好歹咱也是学算法搞竞赛出生的，于是铁了心要搞技术。奈何我被分到了一个加权型班级，懂代码的屈指可数，他们的唯一话题就是分数。在这样一个环境中被孤立，确实很难受。现在好了，技术停滞了，也就想回归“正途”了。

最后，在一个梅雨之夜，我给队长发了条短信，正式决定退出了。

很多人觉得我很傻，很多人觉得我很幼稚，还有很多人更是慷慨的献出的鄙视与嘲讽。我都不在乎，我就想失恋后的醉汉，一点都不在乎，只想找到一瓶酒来继续麻痹自己。我给自己找了很多理由——不适应环境、不喜欢氛围、影响学习……真正的原因我不知道，但本能要我这样做。我失败过很多次，我也学会了在适当的时候选择去失败。

然后呢？我也不知道，彻底变成一个迷途的小丑，在继续寻找归途。

二

很平凡的，走到的期末，我又复活了。

人就是这样，好了伤口忘了痛，但在睡梦中忘记了过去，我也再次变得不知天高地厚了。我给暑假加上了程，参加创新计划夏令营，报考网工，学习web，回归WOW……

起初一切也很完美，我顺利通过了夏令营的测试，开始了救赎之旅。

这确实是个好的选择，在这样一个炎热的暑假，和哥们一起起床，到面包房买上早餐，跑到创新计划团队代码，逗逗那只顽皮的猫，然后一跑到东校门外点上烧烤，喝上几瓶，互相搀扶着，说着大话，吹着牛嘻回到宿舍，在打开电脑，开始艾泽拉斯之旅。

这才是我一直期盼的大学生活啊，很充实，很完美。和志同道合的兄弟，一起挥洒汗水，一起在雨中狂奔狂的大叫，一起享受着青春。

我真的很渴望，再这样来一次。

当然，美好的总是开头，后面就开始继续深陷“大灾变的漩涡”（WOW第三个资料片）里。

三

大二的生活也如约而至了，开学便抛下开发时间被压缩到一半的软酷实训中，本来计划好的开学网工预习也作废了。

软酷确实败了，败在了自大与懒散中。自大在以为以一己之力就能毫无压力的搞定，懒散在确实能搞定于是——你也懂得。虽然完成品还不错，赖和却被无情淘汰。

后来看过获奖作品后确实深受启发。由于以前习惯于竞赛，作品被要求的是以最高的效率完成题目要求。我做到了，我们的代码质量确实可以看到，但我却忘了，评委是看不到的。他们能看到的是什么，UI、创意……

我很懊恼，也很后悔。

走出了一个圈，拖着战败之躯，我突然又想寻找战友了，我也很寂寞，我真的好想找朋友。

这时联创又招新了。

四

对于现在的我，联创的测试确实不难了。同样的测试，和大一时比起来，确实感觉自己有了很多的进步。

自己学了很多东西，C++、JAVA、C#、ASP.NET、SQL等，虽然都只摸到了皮毛，但获得了一个很好的宏观概念，我想我也该选着一条路走下去了吧。

联创的氛围确实不错，这确实很像我期待的那种学生团体。有不同的牛人、不同的思想、不同的味道。可以随意的包夜、随意的学习、随意的聊天。当然，可以看出，大家都有一样的眼神。

还记得小时候看过讲述比尔盖茨和乔布斯的《硅谷传奇》，我大概就是在羡慕他们那种找的挚友然后燃烧生命的生活吧。我很喜欢新东西，特别是电子设备，小学时无意学习了编程然后认识了这个计算机的世界。但这样一个世界却让我寂寞，变得和其他人不一样，出了魔兽世界和考试，就没有共同话题。所以我一直在渴望着什么。大学报了软件，想填补心中饿漏洞，却发现漏洞太大了，同样的专业却没找到同样的人。

在联创半年了，我期待着。

五

大概是真觉得非专业学习什么的无所谓了吧，我迎来了人生第一次挂科。我依然期待着大一那般24小时从入门到精通，结果败了，讨厌的感觉。

那是在其中，第一次迎来考试周，满满的五门课，基本没学过，开始考前突击。依旧我行我素，于是第一门败了，还好总结了教训，警惕着，后面的都过了。

尤记那天考完第一门我就病了，一连发烧了6天，我却没了感觉，只想对着雨笑啊。

最后

看完自己的回忆，我真想笑啊。

原来自己是如此的幼稚，如此的天真。如孩童般随着性子，做着梦，用兴趣选择方向，用感觉赌博未来。

后悔吗？

不。

这就是我的选择，既然走过来了，就会走下去。

回忆着伤痛，一阵冷战，却隐忍着泪水。

2011.
